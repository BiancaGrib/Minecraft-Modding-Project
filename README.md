# Minecraft-Modding-Project

Garthis is a personal Minecraft Java modding project developed with Forge in IntelliJ IDEA. The project currently includes a custom item called Garthium and a dedicated creative tab for organizing mod content. It was created as a hands-on project to explore Java, Forge modding, Gradle configuration, external libraries, and GitHub-based project management.

## Current features
- Custom in-game item: **Garthium**
- Dedicated custom **creative tab**
- Project configured with **Forge**, **Gradle**, and external libraries
- Developed in **IntelliJ IDEA**


  ## Where to find the added content
The current custom content can be found in-game inside the **Garthis creative tab**.

## Technologies used
- Java
- Minecraft Forge
- Gradle
- IntelliJ IDEA
- GitHub

## Project status
This is an ongoing personal project and will be expanded with additional items, blocks, and gameplay features.
